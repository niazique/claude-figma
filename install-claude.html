Установка Claude Code на Mac, в терминале

1
Создай рабочую папку

Все проекты удобнее хранить в одной папке — так Claude всегда знает, где искать файлы, и ничего не потеряется. Чаще всего эту папку называют vibe и располагают прямо на рабочем столе — так до неё проще всего добраться.

2
Открой терминал

Все команды ниже вводятся в терминале. На Mac он называется Terminal — найди его через Spotlight: нажми ⌘ Space, введи «Terminal» и нажми Enter. Или открой через Finder → Программы → Утилиты → Terminal.

Каждую команду вводи и нажимай Enter для выполнения.

3
Установи Homebrew

Менеджер пакетов для Mac. Пропусти, если уже установлен.

В процессе установки терминал попросит ввести пароль от Mac — это обычный запрос прав администратора. Вводимые символы не будут отображаться в строке, но они фиксируются — просто введи пароль и нажми Enter.

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
4
Установи Node.js
brew install node

Установка обычно занимает 1–3 минуты. Когда завершится, убедись, что всё прошло успешно — введи команду:

node -v

Если в ответ появилась версия — всё готово.

5
Установи Claude Code
npm install -g @anthropic-ai/claude-code

Обычно занимает меньше минуты.

6
Добавь алиасы ETG

Алиас — это короткое слово-команда, которое заменяет длинную строку с настройками. После добавления можно просто написать etg в терминале — и Claude запустится с нужными параметрами автоматически.

Ключ можно получить на chat.etg.team — войди через Keycloak, нажми на иконку профиля в правом верхнем углу, перейди в настройки аккаунта. В левой колонке выбери Аккаунт — в самом низу будет раздел с API ключами. Создай новый ключ, он автоматически скопируется в буфер обмена.

Выполни обе команды по очереди:

echo 'alias etg='"'"'cd ~/Desktop/vibe && ANTHROPIC_AUTH_TOKEN="API_KEY_HERE" ANTHROPIC_BASE_URL="https://llmgate.etg.team" ANTHROPIC_MODEL="bedrock-claude-4-6-sonnet" ANTHROPIC_DEFAULT_OPUS_MODEL="bedrock-claude-4-6-opus" ANTHROPIC_DEFAULT_SONNET_MODEL="bedrock-claude-4-6-sonnet" ANTHROPIC_DEFAULT_HAIKU_MODEL="bedrock-claude-4-5-haiku" CLAUDE_CODE_ENABLE_TELEMETRY="0" CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC="1" CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS="1" DISABLE_PROMPT_CACHING_OPUS="1" DISABLE_PROMPT_CACHING_SONNET="1" DISABLE_PROMPT_CACHING_HAIKU="1" claude'"'"'' >> ~/.zshrc
echo 'alias etgx='"'"'cd ~/Desktop/vibe && ANTHROPIC_AUTH_TOKEN="API_KEY_HERE" ANTHROPIC_BASE_URL="https://llmgate.etg.team" ANTHROPIC_MODEL="bedrock-claude-4-6-sonnet" ANTHROPIC_DEFAULT_OPUS_MODEL="bedrock-claude-4-6-opus" ANTHROPIC_DEFAULT_SONNET_MODEL="bedrock-claude-4-6-sonnet" ANTHROPIC_DEFAULT_HAIKU_MODEL="bedrock-claude-4-5-haiku" CLAUDE_CODE_ENABLE_TELEMETRY="0" CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC="1" CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS="1" DISABLE_PROMPT_CACHING_OPUS="1" DISABLE_PROMPT_CACHING_SONNET="1" DISABLE_PROMPT_CACHING_HAIKU="1" claude --dangerously-skip-permissions'"'"'' >> ~/.zshrc

Важно: перезапусти терминал — закрой и открой заново.

7
Запусти Claude

Теперь для запуска Claude достаточно написать одну из двух команд:

КомандаОписание
etg Обычный режим — Claude спрашивает разрешение перед каждым действием
etgx Опасный режим — не спрашивает подтверждений и делает то, что считает нужным.
Запускай только когда хорошо разобрался, как всё работает: без присмотра Claude может навредить проекту или устройству.
8
Подключи Figma к Claude Code

Установи официальный Figma плагин для Claude Code — он добавляет MCP сервер и скиллы для работы с дизайном.

MCP-сервер — это расширение, которое даёт Claude доступ к внешним инструментам. В данном случае — к Figma: Claude сможет читать дизайн, получать размеры, цвета и структуру компонентов прямо из файла.

Скиллы — это готовые инструкции для Claude, которые описывают, как решать конкретные задачи. Например, скилл для Figma подсказывает, как правильно работать с дизайном: что запрашивать, как интерпретировать и как переводить в код.

Шаг 1. Установи плагин:

claude plugin install figma@claude-plugins-official

Шаг 2. Перезапусти Claude Code если он был открыт, затем введи команду:

/plugin

В открывшемся меню перейди во вкладку Installed → выбери figma → нажми Enter → откроется страница авторизации → нажми Allow access.

После этого статус плагина сменится на connected — всё готово.

9
Включи MCP-сервер в Figma

Чтобы Claude мог читать твои дизайны напрямую, нужно включить MCP-сервер внутри самой Figma — это делается один раз.

Шаг 1. Открой любой файл в Figma.

Шаг 2. Внизу экрана на панели инструментов найди переключалку на дев режим с иконкой </> — нажми на неё, чтобы включить Dev Mode.

Шаг 3. В правой панели найди раздел MCP Server и включи его — переключатель должен стать активным.

После этого Claude сможет обращаться к Figma и читать структуру, стили и компоненты твоих файлов.

🎉

Всё готово, можно работать!

Claude подключён к Figma, алиасы настроены, плагин авторизован. Теперь достаточно открыть терминал, написать etg — и можно вайбкодить.

Добавь функции ETG

На Windows вместо алиасов используются функции PowerShell — принцип тот же: вводишь etg и Claude запускается с нужными параметрами.

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Частые вопросы

У меня уже есть личная подписка на Claude — как использовать ETG для работы?

Личная подписка и рабочий доступ через ETG — это разные вещи, они не мешают друг другу. Если просто запустить claude в терминале — он будет использовать твою личную подписку. А если запустить через etg — он подключится к серверу ETG с рабочим ключом.

Алиасы etg / etgx как раз для этого и нужны: они автоматически подставляют рабочий ключ и сервер, чтобы ты не путал личное и рабочее. Для рабочих задач — всегда запускай через etg.

Как запустить Claude Code в VS Code или другом редакторе?

Claude Code работает в терминале — точно так же, как и в обычной системе. Чтобы запустить его внутри редактора, достаточно открыть встроенный терминал: в VS Code это Ctrl + ` (Mac и Windows). В появившемся терминале введи etg или etgx — и всё будет работать как обычно.

Figma просит авторизацию — что делать?

Если при работе с Figma Claude пишет, что нужна авторизация — введи /mcp в Claude. В списке серверов найди figma — рядом будет ссылка для авторизации. Открой её в браузере и разреши доступ.

После авторизации нажми Esc и повтори свой запрос — это очень важно, без повторного запроса Claude не начнёт работу с Figma.

Скиллы сжигают много токенов — что делать?

Да, скиллы иногда способствуют сжиганию большого количества токенов — особенно если в них много инструкций, которые тебе не нужны. Если хочешь, всегда можно попробовать модифицировать скиллы под себя (убрать лишнее) или написать свои с нуля.

Скиллы — это просто текстовые инструкции. Ничего сложного. А о том, как их найти и отредактировать — спроси у Claude сам 🙂

Есть ли лимиты на использование?

Да, лимиты есть. К сожалению, посмотреть их нигде нельзя — ни в интерфейсе, ни в настройках. Когда лимит исчерпан, Claude сообщит об этом сам.

Быстрее всего лимит расходуется при работе с моделью Opus — она самая мощная, но и самая «дорогая» с точки зрения квоты. Для простых задач лучше переключиться на Sonnet — он справляется с большинством повседневных запросов и расходует лимит значительно медленнее. Лимиты обновляются раз в месяц.

Claude завис или не отвечает

Нажми Esc, чтобы прервать текущее действие. Если не помогает — нажми Ctrl + C дважды, чтобы полностью выйти, и запусти заново через etg.

Как обновить Claude Code?

Выполни ту же команду, что и при установке:

npm install -g @anthropic-ai/claude-code

Она установит последнюю версию поверх текущей.

Можно ли использовать Claude без алиасов?

Да, но тогда перед каждым запуском нужно вручную задавать переменные окружения — ключ, адрес сервера, модель. Алиасы etg / etgx просто делают это автоматически в одну команду.

Почему в названии модели написано «bedrock»? Это Amazon?

Да, Amazon Bedrock — это облачная платформа, через которую ETG получает доступ к модели Claude от Anthropic. Это просто способ доставки — как Amazon доставляет посылки, так Bedrock доставляет ИИ-модели.

Для тебя это ни на что не влияет: не нужен аккаунт Amazon, не нужна подписка. Всё работает через сервер ETG — ты просто пишешь etg и общаешься с Claude.

Устал печатать — можно голосом?

Да! Установи Wispr Flow — это приложение для голосового ввода, которое работает в любом текстовом поле, включая терминал. Просто говоришь — и текст появляется.

Бесплатной версии вполне хватает для повседневной работы. Скачай с wispr.ai, установи и пользуйся.

Как выйти из Claude?

Нажми Esc, чтобы прервать текущее действие. Чтобы полностью выйти из Claude, введи /exit или нажми Ctrl + C дважды.
